## Commands

```javascript

// include node_modules packages
yarn

// Start server http://localhost:3000
yarn dev

// Build dist
yarn build

// Fix all ts,json,js,tsx prettier
yarn fix:prettier

// Jest test
yarn test

// Jest coverage
yarn test:coverage

// E2E test (videos created in map ./test-results)
yarn test:e2e

```
